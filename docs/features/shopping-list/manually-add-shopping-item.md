# Manually Add a Shopping Item

## Feature Area

Shopping List

## User Story

As a home cook, I want to manually add items to my shopping list so that I can keep track of things I need to buy.

## Goal

Give users a basic shopping list they can use at the store, even before recipe or pantry features are connected. The list should be easy to add to and easy to read.

## Acceptance Criteria

- The user can view their shopping list.
- The user can add a new item by entering an item name.
- The user can optionally enter a quantity or short note (for example, `1 gallon` or `organic`).
- After adding an item, it appears on the shopping list as not yet purchased.
- The user can add multiple items in one session without leaving the list.
- If the user tries to add an item without a name, the app shows a clear error message.

## Notes

- Keep the first version focused on manual entry only.
- Future improvements: check off purchased items, add items from recipes, add low-stock pantry items, group items by store aisle.
- This story is independently useful and does not require recipes or pantry data to work.
