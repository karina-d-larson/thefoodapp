# Create a Recipe Manually

## Feature Area

Recipe Library

## User Story

As a home cook, I want to create a recipe with a title and ingredients so that I can save meals I cook regularly.

## Goal

Let users build their own recipe collection inside the app. A saved recipe becomes something they can view, edit, and use in meal planning and shopping later.

## Acceptance Criteria

- The user can open a form to create a new recipe.
- The user must enter a recipe title before saving.
- The user can add one or more ingredients, each with a name and optional amount (for example, `2 cups flour`).
- After saving, the new recipe appears in the recipe library.
- If the user tries to save without a title, the app shows a clear error message.
- The user can cancel creating a recipe without saving changes.

## Notes

- Instructions, notes, folders, and search can be added in separate stories.
- Future improvements: import a recipe from a web link, duplicate an existing recipe, add photos.
- This story does not require pantry or shopping list features to be complete first.
